# 🚀 AI-Based Autonomous Navigation System

## 📌 Project Overview
This project simulates an AI-based autonomous navigation system where a robot finds the shortest path from a start point to a destination while avoiding obstacles.

## 🎯 Objective
To design a virtual robot navigation system using AI algorithms like A* path planning without using physical hardware.

## 🧠 Features
- Start and End point selection
- Obstacle creation and removal
- A* Path Planning Algorithm
- Real-time robot movement
- Grid-based simulation using Pygame

## 🛠️ Tech Stack
- Python
- Pygame
- A* Algorithm

| Action | Key |
|------|-----|
| Select Start | S |
| Select End | E |

| Add Obstacle | Left Click |
| Remove Obstacle | Right Click |
| Reset | R |
