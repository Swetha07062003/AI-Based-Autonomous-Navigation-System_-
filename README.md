# AI-Based Autonomous Navigation System

## Project Overview
This project shows a robot moving from start to end using AI path planning.

## Features
- Start and End selection
- Obstacle placement
- A* path planning
- Robot movement

## How to Run
python main.py

## Controls
S → Select Start  
E → Select End  
SPACE → Start Robot  
Left Click → Add obstacle  
Right Click → Remove obstacle  
R → Reset
